# Demo video

Place your recorded demo video here, e.g. `founder-school-agent-demo.mp4`.

Then link it from the **Demo Video** section of the [root README](../../README.md):

```markdown
**[▶️ Watch the demo](./docs/demo/founder-school-agent-demo.mp4)**
```

The narrated voice-over script and shot list are in [DEMO.md](../../DEMO.md).

> Tip: GitHub does not autoplay an MP4 linked from the README. For inline playback, drag the file into
> a GitHub issue/PR/release (GitHub hosts it and gives an embeddable URL), or upload to YouTube / Loom
> and link that instead.
